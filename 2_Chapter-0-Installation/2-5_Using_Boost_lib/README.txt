Using Custom library | Boost C++
* Download from here:
	- https://www.boost.org/
* Extract the "boost" folder from the downloaded zip folder to your "../c++" location of mingw-64:
	- "C:\Program Files\mingw-w64\x86_64-8.1.0-posix-seh-rt_v6-rev0\mingw64\lib\gcc\x86_64-w64-mingw32\8.1.0\include\c++"
* Also, copy the folder from here to this location as well via bash-terminal (Ubuntu):
	- `sudo cp boost /usr/local/include -r`
	- "/usr/local/include"
